# Project Lab Data Processing

Supplementary code for Data Processing Pipeline used for Project Lab Report of C. Wardell at TU Graz.

## Usage

#### 1. Setup venv & install requirements.txt
#### 2. Run data processing pipeline:
```bash
python3 scripts/multiple_file_analysis.py
```

Final processing results are inside:
`data/out/result.xlsx`.

#### 3. Visualize Data:

```bash
python3 scripts/plot_results.py 
```
